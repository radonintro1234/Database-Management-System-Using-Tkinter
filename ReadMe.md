# Database management system using Tkinter
This is a MY BOOKS app build using python.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install prerequisite.

```bash
pip install tkinter
pip install sqlite3
pip install pandas
pip install fpdf
```

## Usage

```bash
$ cd src
$ python Authentication.py
```
